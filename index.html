WEB DEVELOPMENT SKILLS

Learn html

HTML stands for Hyper Text Markup Language. HTML is the content of web page. Examples of HTML include headings, paragraph, image, links, list, etc.

Check out the HTML tutorial on w3schools.com

Learn Css

CSS stands for Cascading Style Sheets. CSS is the style of a web page. Examples of CSS include font styles and color, background color, borders, dimensions, layout, etc.

Check out the CSS tutorial on w3schools.com

Learn javascript

Javascript is the programing language of the web. Javascript helps to make web pages interactive and dynamic. Some common javascript rasks are:

  • add new HTML to the pages
  • Change the existing content / style
  • react to user events (mouse, clicks, keypresses, etc.)

Check out the javascript tutorial on javascript.info.
